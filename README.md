# Gemini SQL Retriever

A simple Streamlit app that turns English questions into SQL queries using Google Gemini AI and runs them on a SQLite database.

## Steps to Set Up and Run

1. Clone the project folder to your local machine.

2. Install required Python packages.

3. Create a `.env` file in the project directory and add your Google API key:
GOOGLE_API_KEY=your_api_key_here

5. Run the app with the command:
streamlit run app.py

6. In the app, type your question in English and click the button.

7. View the generated SQL query and query results displayed in the app.
